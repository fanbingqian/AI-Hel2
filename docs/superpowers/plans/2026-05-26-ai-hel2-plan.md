# AI-Hel2 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build AI-Hel2 — a Tauri desktop AI assistant with 3D knowledge sphere, voice chat, document editor, and Excalidraw canvas.

**Architecture:** Four-page Tauri 2.0 app. Rust backend **services layer copied from AI-Hel** (agent_manager, knowledge_service, wiki_service, session_service, etc.) — same business logic, same Python Hermes Agent and Heimdall. **Commands layer written new** as invoke adapters. React 19 frontend with Three.js/R3F renders 3D sphere + particle animations driven by Web Audio API. Frontend communicates with Rust via Tauri invoke() + SSE events.

**完整通信链路:**

```
React 前端                              Tauri Rust 后端                   外部服务
───────────                             ────────────────                 ──────────
invoke("chat_send")  ──────────────→   commands/chat.rs
                                        │→ AgentManager
                                        │→ HTTP SSE reqwest ──────────→ Python Hermes Agent
                                        │                                 http://127.0.0.1:8642
emit("chat:delta")  ←────────────────│← SSE stream response            (对话推理 + TTS)
listen("chat:delta")

invoke("graph_data") ──────────────→   commands/knowledge.rs
                                        │→ KnowledgeService ───────────→ Heimdall 知识服务
                                        │  (SQLite + 向量检索)            http://127.0.0.1:8765
GraphData          ←────────────────│← 查询结果

invoke("get_wiki_file_tree") ──────→   commands/wiki.rs
                                        │→ WikiService ────────────────→ 本地文件系统
                                        │  (文件读写)                    ~/heimdall/wiki/
FileNode[]         ←────────────────│← 文件树

invoke("canvas_save") ─────────────→   commands/canvas.rs ────────────→ 本地文件系统
```

```rust
use crate::services::agent_manager::AgentManager;
use crate::services::connection_service::ConnectionService;
use tauri::{AppHandle, Emitter, Manager, State};
use serde::{Deserialize, Serialize};
use std::sync::Arc;
use std::sync::atomic::{AtomicBool, Ordering};

#[derive(Clone, Serialize)]
pub struct StreamDelta {
    pub content: String,
}

#[derive(Clone, Serialize)]
pub struct StreamDone {
    pub total_tokens: u32,
}

#[derive(Clone, Serialize)]
pub struct StreamError {
    pub message: String,
}

pub struct ChatState {
    pub agent: AgentManager,
    pub cancel_flag: Arc<AtomicBool>,
}

impl ChatState {
    pub fn new(connection: ConnectionService) -> Self {
        Self {
            agent: AgentManager::new(connection),
            cancel_flag: Arc::new(AtomicBool::new(false)),
        }
    }
}

#[derive(Deserialize)]
pub struct ChatRequest {
    pub message: String,
    pub session_id: Option<String>,
}

#[tauri::command]
pub async fn chat_send(
    app: AppHandle,
    state: State<'_, ChatState>,
    request: ChatRequest,
) -> Result<(), String> {
    state.cancel_flag.store(false, Ordering::SeqCst);
    let agent = &state.agent;

    let stream = agent
        .chat_stream(&request.message, request.session_id.as_deref())
        .await
        .map_err(|e| format!("Failed to start chat: {e}"))?;

    // Stream handling will be added when porting hermes.rs fully
    // For now emit a placeholder delta
    let _ = app.emit("chat:delta", StreamDelta {
        content: "Hello from AI-Hel2".into(),
    });
    let _ = app.emit("chat:done", StreamDone { total_tokens: 1 });

    Ok(())
}
```

```rust
pub async fn canvas_open(
    state: tauri::State<'_, CanvasCommandsState>,
    path: String,
) -> Result<CanvasData, String> {
    let base = state.base_dir.lock().map_err(|e| e.to_string())?;
    let full = std::path::Path::new(&*base).join(&path);
    let content = std::fs::read_to_string(&full)
        .map_err(|e| format!("Failed to read canvas file: {e}"))?;
    serde_json::from_str(&content)
        .map_err(|e| format!("Failed to parse canvas: {e}"))
}

#[tauri::command]
pub async fn canvas_save(
    state: tauri::State<'_, CanvasCommandsState>,
    path: String,
    data: CanvasData,
) -> Result<(), String> {
    let base = state.base_dir.lock().map_err(|e| e.to_string())?;
    let full = std::path::Path::new(&*base).join(&path);
    if let Some(parent) = full.parent() {
        std::fs::create_dir_all(parent).map_err(|e| format!("Failed to create dir: {e}"))?;
    }
    let content = serde_json::to_string_pretty(&data)
        .map_err(|e| format!("Failed to serialize: {e}"))?;
    std::fs::write(&full, content).map_err(|e| format!("Failed to write canvas: {e}"))
}
```

### Task 5: Wire Rust backend — lib.rs + commands + swappable-agent architecture

**Design principle:** Hermes Agent is a **replaceable black-box HTTP API**. All knowledge injection,
entity extraction, and session management happen in the Rust backend. The Agent only receives
OpenAI-compatible `/v1/chat/completions` requests. To swap agents, change the URL in
`ConnectionService` — zero code changes elsewhere.

**Architecture: Push + Pull knowledge model:**

```
┌─ Push Model (P1.3) ──────────────────────────────────────────────────┐
│  Before chat: Rust backend calls build_context_snapshot()             │
│  → queries local SQLite cache + Heimdall for relevant knowledge      │
│  → formats as system message (≤800 chars)                            │
│  → prepends to message array → sends to Hermes Agent                 │
│  Agent NEVER knows about knowledge graphs.                            │
├─ Pull Model (Entity Extraction) ─────────────────────────────────────┤
│  After chat:done event: frontend concatenates user+assistant text     │
│  → calls extract_entities_from_text()                                │
│  → sends to Heimdall /api/extract (LLM extraction)                   │
│  → falls back to local regex ([[wikilinks]], 《书名》, **粗体**)      │
└──────────────────────────────────────────────────────────────────────┘
```

**Hermes Agent independence & swappability:**

```
                       Rust Backend (glue layer)
┌──────────┐          ┌──────────────────────────┐          ┌──────────────────┐
│  React   │ invoke() │  chat.rs                  │ HTTP SSE │  Hermes Agent    │
│  frontend│────────→ │  ┌──────────────────────┐ │────────→ │  (black box)     │
│          │←──────── │  │ 1. build_context()   │ │←──────── │  /v1/chat/       │
│          │ events   │  │ 2. inject as system  │ │  stream  │  completions     │
│          │          │  │ 3. POST /v1/chat     │ │          │                  │
│          │          │  └──────────────────────┘ │          │  Replace with:   │
│          │          │                           │          │  - OpenAI API    │
│          │          │  knowledge.rs             │ HTTP     │  - Local LLM     │
│          │          │  ┌──────────────────────┐ │────────→ │  - Claude API    │
│          │          │  │ extract_entities()   │ │←──────── │  - Any OpenAI-   │
│          │          │  │ get_graph_data()     │ │  JSON    │    compatible    │
│          │          │  └──────────────────────┘ │          │    endpoint      │
│          │          │                           │          └──────────────────┘
│          │          │  ConnectionService        │
│          │          │  ┌──────────────────────┐ │
│          │          │  │ agent_url() → config  │ │  ← Change this URL to swap agents
│          │          │  │ default: 127.0.0.1:   │ │
│          │          │  │          8642         │ │
│          │          │  └──────────────────────┘ │
└──────────┘          └──────────────────────────┘
```

```typescript
import { create } from "zustand";
import type { ChatMessage } from "../types/chat";
import { invoke } from "@tauri-apps/api/core";
import { listen } from "@tauri-apps/api/event";

function genId(): string {
  return Date.now().toString(36) + Math.random().toString(36).slice(2);
}

interface ChatState {
  messages: ChatMessage[];
  isLoading: boolean;
  sessionId: string | null;
  addMessage: (role: "user" | "assistant", content: string) => void;
  sendMessage: (content: string) => Promise<void>;
  setupListeners: () => Promise<() => void>;
}

export const useChatStore = create<ChatState>((set, get) => ({
  messages: [],
  isLoading: false,
  sessionId: null,

  addMessage: (role, content) => {
    const msg: ChatMessage = {
      id: genId(),
      role,
      content,
      timestamp: Date.now(),
    };
    set((s) => ({ messages: [...s.messages, msg] }));
  },

  sendMessage: async (content: string) => {
    const { addMessage, sessionId } = get();
    addMessage("user", content);
    set({ isLoading: true });

    // Start streaming response placeholder
    addMessage("assistant", "");

    try {
      await invoke("chat_send", { request: { message: content, sessionId } });
    } catch (e) {
      set({ isLoading: false });
    }
  },

  setupListeners: async () => {
    const unlistenDelta = await listen<{ content: string }>("chat:delta", (evt) => {
      set((s) => {
        const msgs = [...s.messages];
        const last = msgs[msgs.length - 1];
        if (last && last.role === "assistant") {
          msgs[msgs.length - 1] = {
            ...last,
            content: last.content + evt.payload.content,
          };
        }
        return { messages: msgs };
      });
    });

    const unlistenDone = await listen<{ total_tokens: number }>("chat:done", () => {
      set({ isLoading: false });
    });

    return () => {
      unlistenDelta();
      unlistenDone();
    };
  },
}));
```

```typescript
import { useMemo } from "react";
import { forceSimulation, forceCollide, forceManyBody } from "d3-force";
import type { Entity, Relation } from "../types/knowledge";

export interface SphereNode {
  id: string;
  name: string;
  entityType: string;
  confidence: number;
  position: [number, number, number];  // x, y, z on sphere surface
  radius: number;  // node sphere radius
}

const SPHERE_R = 2.5;  // sphere radius in scene units

function fibonacciSphere(n: number, total: number): [number, number, number] {
  const phi = Math.acos(1 - 2 * (n + 0.5) / total);
  const theta = Math.PI * (1 + Math.sqrt(5)) * n;
  return [
    SPHERE_R * Math.sin(phi) * Math.cos(theta),
    SPHERE_R * Math.cos(phi),
    SPHERE_R * Math.sin(phi) * Math.sin(theta),
  ];
}

function nodeRadius(entityType: string): number {
  const t = entityType.toLowerCase();
  if (t.includes("concept") || t.includes("core")) return 0.08;
  if (t.includes("document") || t.includes("reference")) return 0.03;
  return 0.05;
}

export function useSphereNodes(entities: Entity[], relations: Relation[]): SphereNode[] {
  return useMemo(() => {
    if (entities.length === 0) return [];

    const total = entities.length;
    const nodes: SphereNode[] = entities.map((e, i) => ({
      id: e.id,
      name: e.name,
      entityType: e.entity_type,
      confidence: e.confidence,
      position: fibonacciSphere(i, total),
      radius: nodeRadius(e.entity_type),
    }));

    // Apply D3 spherical force to pull related nodes closer
    const sim = forceSimulation(
      nodes.map((n) => ({ ...n, x: n.position[0], y: n.position[1], z: n.position[2] })),
    )
      .force("collide", forceCollide(0.1))
      .force("charge", forceManyBody().strength(-0.05))
      .alphaDecay(0.02);

    // Add link forces — pull related nodes closer on the sphere
    const linkForce = relations.map((r) => ({
      source: r.source_id,
      target: r.target_id,
    }));

    // Simple approach: just run a few ticks
    sim.stop();

    // Re-project to sphere surface
    nodes.forEach((n) => {
      const len = Math.sqrt(
        n.position[0] ** 2 + n.position[1] ** 2 + n.position[2] ** 2,
      );
      if (len > 0) {
        n.position = [
          (n.position[0] / len) * SPHERE_R,
          (n.position[1] / len) * SPHERE_R,
          (n.position[2] / len) * SPHERE_R,
        ];
      }
    });

    return nodes;
  }, [entities, relations]);
}
```

```tsx
import { useAudioStore } from "../../stores/audioStore";
import * as THREE from "three";
import type { SphereNode } from "../../hooks/useSphereNodes";

interface Props {
  nodes: SphereNode[];
}

const PARTICLE_COUNT = 2000;

export function AudioParticles({ nodes }: Props) {
  const pointsRef = useRef<THREE.Points>(null);
  const volume = useAudioStore((s) => s.volume);
  const spectrum = useAudioStore((s) => s.spectrum);

  const { positions, basePositions, colors } = useMemo(() => {
    const pos = new Float32Array(PARTICLE_COUNT * 3);
    const base = new Float32Array(PARTICLE_COUNT * 3);
    const col = new Float32Array(PARTICLE_COUNT * 3);

    for (let i = 0; i < PARTICLE_COUNT; i++) {
      // Distribute particles around nodes
      const nodeIdx = Math.floor(Math.random() * Math.max(nodes.length, 1));
      const node = nodes[nodeIdx] || { position: [0, 0, 0] as [number, number, number], radius: 0.05 };

      const phi = Math.random() * Math.PI * 2;
      const theta = Math.random() * Math.PI;
      const r = node.radius + Math.random() * 0.3;

      pos[i * 3] = node.position[0] + r * Math.sin(theta) * Math.cos(phi);
      pos[i * 3 + 1] = node.position[1] + r * Math.cos(theta);
      pos[i * 3 + 2] = node.position[2] + r * Math.sin(theta) * Math.sin(phi);

      base[i * 3] = pos[i * 3];
      base[i * 3 + 1] = pos[i * 3 + 1];
      base[i * 3 + 2] = pos[i * 3 + 2];

      // Green gradient to white
      const t = Math.random();
      col[i * 3] = 0.15 + t * 0.85;       // R: green → white
      col[i * 3 + 1] = 0.78 + t * 0.22;   // G
      col[i * 3 + 2] = 0.48 + t * 0.52;   // B
    }

    return { positions: pos, basePositions: base, colors: col };
  }, [nodes]);

  useFrame((_, delta) => {
    if (!pointsRef.current) return;
    const posArr = pointsRef.current.geometry.attributes.position.array as Float32Array;

    for (let i = 0; i < PARTICLE_COUNT; i++) {
      const freqIdx = Math.floor((i / PARTICLE_COUNT) * spectrum.length);
      const freqVal = spectrum[freqIdx] / 255; // 0-1

      // Displace outward along normal from sphere center
      const bx = basePositions[i * 3];
      const by = basePositions[i * 3 + 1];
      const bz = basePositions[i * 3 + 2];
      const len = Math.sqrt(bx * bx + by * by + bz * bz) || 1;
      const nx = bx / len;
      const ny = by / len;
      const nz = bz / len;

      const displace = freqVal * 0.5 * (1 + volume);
      const springTarget = displace * 0.8;

      posArr[i * 3] = bx + nx * (springTarget + (Math.random() - 0.5) * 0.02);
      posArr[i * 3 + 1] = by + ny * (springTarget + (Math.random() - 0.5) * 0.02);
      posArr[i * 3 + 2] = bz + nz * (springTarget + (Math.random() - 0.5) * 0.02);
    }

    pointsRef.current.geometry.attributes.position.needsUpdate = true;
  });

  return (
    <points ref={pointsRef}>
      <bufferGeometry>
        <bufferAttribute
          attach="attributes-position"
          count={PARTICLE_COUNT}
          array={positions}
          itemSize={3}
        />
        <bufferAttribute
          attach="attributes-color"
          count={PARTICLE_COUNT}
          array={colors}
          itemSize={3}
        />
      </bufferGeometry>
      <pointsMaterial
        size={0.02}
        vertexColors
        transparent
        depthWrite={false}
        blending={THREE.AdditiveBlending}
      />
    </points>
  );
}
```

```typescript
import { useRef, useCallback, useState } from "react";

const WAKE_WORD = "hi hel";
const SILENCE_TIMEOUT = 5000; // ms

interface UseVoiceInputOptions {
  onResult: (text: string) => void;
  onWake: () => void;
}

export function useVoiceInput({ onResult, onWake }: UseVoiceInputOptions) {
  const [isListening, setIsListening] = useState(false);
  const recognitionRef = useRef<any>(null);
  const silenceTimerRef = useRef<ReturnType<typeof setTimeout> | null>(null);
  const fullTranscriptRef = useRef("");

  const resetSilenceTimer = useCallback(() => {
    if (silenceTimerRef.current) clearTimeout(silenceTimerRef.current);
    silenceTimerRef.current = setTimeout(() => {
      const text = fullTranscriptRef.current.trim();
      if (text) onResult(text);
      fullTranscriptRef.current = "";
      setIsListening(false);
      recognitionRef.current?.stop();
    }, SILENCE_TIMEOUT);
  }, [onResult]);

  const startListening = useCallback(() => {
    const SpeechRecognition =
      (window as any).SpeechRecognition || (window as any).webkitSpeechRecognition;
    if (!SpeechRecognition) {
      console.warn("SpeechRecognition not available");
      return;
    }

    const recognition = new SpeechRecognition();
    recognition.lang = "zh-CN";
    recognition.continuous = true;
    recognition.interimResults = true;
    recognitionRef.current = recognition;

    recognition.onresult = (event: any) => {
      let interim = "";
      for (let i = event.resultIndex; i < event.results.length; i++) {
        const transcript = event.results[i][0].transcript.trim().toLowerCase();
        if (!isListening && transcript.includes(WAKE_WORD)) {
          onWake();
          setIsListening(true);
          fullTranscriptRef.current = "";
          continue;
        }
        if (isListening) {
          interim += event.results[i][0].transcript;
        }
      }
      fullTranscriptRef.current = interim;
      resetSilenceTimer();
    };

    recognition.onerror = () => {
      setIsListening(false);
    };

    recognition.onend = () => {
      // Auto restart for continuous wake word detection
      if (!isListening) {
        setTimeout(() => recognition.start(), 200);
      }
    };

    recognition.start();
  }, [isListening, onWake, resetSilenceTimer]);

  const stopListening = useCallback(() => {
    if (silenceTimerRef.current) clearTimeout(silenceTimerRef.current);
    const text = fullTranscriptRef.current.trim();
    if (text) onResult(text);
    fullTranscriptRef.current = "";
    setIsListening(false);
    recognitionRef.current?.stop();
  }, [onResult]);

  return { isListening, startListening, stopListening };
}
```

```typescript
import { useRef, useCallback, useEffect } from "react";

export function useAudioAnalyzer() {
  const audioCtxRef = useRef<AudioContext | null>(null);
  const analyserRef = useRef<AnalyserNode | null>(null);
  const sourceRef = useRef<MediaStreamAudioSourceNode | null>(null);
  const rafRef = useRef<number>(0);
  const setSpectrum = useAudioStore((s) => s.setSpectrum);
  const setVolume = useAudioStore((s) => s.setVolume);

  const attachToAudioStream = useCallback(
    (stream: MediaStream) => {
      const ctx = new AudioContext();
      const analyser = ctx.createAnalyser();
      analyser.fftSize = 256;
      const source = ctx.createMediaStreamSource(stream);
      source.connect(analyser);

      audioCtxRef.current = ctx;
      analyserRef.current = analyser;
      sourceRef.current = source;

      const dataArray = new Uint8Array(analyser.frequencyBinCount);

      const tick = () => {
        analyser.getByteFrequencyData(dataArray);
        setSpectrum(new Uint8Array(dataArray));
        const avg = dataArray.reduce((a, b) => a + b, 0) / dataArray.length / 255;
        setVolume(avg);
        rafRef.current = requestAnimationFrame(tick);
      };

      tick();
    },
    [setSpectrum, setVolume],
  );

  const disconnect = useCallback(() => {
    cancelAnimationFrame(rafRef.current);
    sourceRef.current?.disconnect();
    audioCtxRef.current?.close();
  }, []);

  useEffect(() => {
    return () => {
      disconnect();
    };
  }, [disconnect]);

  // Hook into TTS output when it plays
  const attachToTTS = useCallback(() => {
    // Create a silent stream that mirrors the audio context output
    // The AnalyserNode taps into the audio graph to extract spectrum
    if (!audioCtxRef.current) {
      const ctx = new AudioContext();
      const analyser = ctx.createAnalyser();
      analyser.fftSize = 256;
      analyser.connect(ctx.destination); // don't mute — pass through

      audioCtxRef.current = ctx;
      analyserRef.current = analyser;

      const dataArray = new Uint8Array(analyser.frequencyBinCount);
      const tick = () => {
        analyser.getByteFrequencyData(dataArray);
        setSpectrum(new Uint8Array(dataArray));
        const avg = dataArray.reduce((a, b) => a + b, 0) / dataArray.length / 255;
        setVolume(avg);
        rafRef.current = requestAnimationFrame(tick);
      };
      tick();
    }
  }, [setSpectrum, setVolume]);

  return { attachToAudioStream, attachToTTS, disconnect };
}
```

```tsx
function formatDateGroup(ts: number): string {
  const now = new Date();
  const d = new Date(ts);
  const diffDays = Math.floor((now.getTime() - d.getTime()) / 86400000);
  if (diffDays === 0) return "今天";
  if (diffDays === 1) return "昨天";
  if (diffDays < 7) return "本周";

}

    filtered.forEach((s) => {
      const key = formatDateGroup(s.lastActiveAt);
      if (!groups[key]) groups[key] = [];
      groups[key].push(s);
```

```tsx
import { useState, useEffect, useCallback } from "react";
import { Editor, rootCtx, defaultValueCtx } from "@milkdown/kit/core";
import { commonmark } from "@milkdown/kit/preset/commonmark";
import { gfm } from "@milkdown/kit/preset/gfm";
import { Milkdown, MilkdownProvider, useEditor } from "@milkdown/react";
import { listener, listenerCtx } from "@milkdown/plugin-listener";
import * as wikiService from "../../services/wiki";

interface Props {
  filePath: string | null;
}

function EditorInner({ content, onChange }: { content: string; onChange: (md: string) => void }) {
  useEditor((root) => {
    return Editor.make()
      .config((ctx) => {
        ctx.set(rootCtx, root);
        ctx.set(defaultValueCtx, content);
        ctx.get(listenerCtx).markdownUpdated((_, md) => onChange(md));
      })
      .use(commonmark)
      .use(gfm)
      .use(listener);
  }, []);

  return <Milkdown />;
}

export function MilkdownEditor({ filePath }: Props) {
  const [content, setContent] = useState("");
  const [dirty, setDirty] = useState(false);

  useEffect(() => {
    if (!filePath) {
      setContent("");
      return;
    }
    wikiService.readFile(filePath).then(setContent).catch(console.error);
    setDirty(false);
  }, [filePath]);

  const handleChange = useCallback((md: string) => {
    setContent(md);
    setDirty(true);
  }, []);

  const handleSave = useCallback(() => {
    if (!filePath || !dirty) return;
    wikiService.writeFile(filePath, content).then(() => setDirty(false));
  }, [filePath, content, dirty]);

  // Auto-save on Ctrl+S
  useEffect(() => {
    const handler = (e: KeyboardEvent) => {
      if ((e.ctrlKey || e.metaKey) && e.key === "s") {
        e.preventDefault();
        handleSave();
      }
    };
    window.addEventListener("keydown", handler);
    return () => window.removeEventListener("keydown", handler);
  }, [handleSave]);

  return (
    <div style={{ flex: 1, display: "flex", flexDirection: "column", overflow: "hidden" }}>
      <div style={{ display: "flex", alignItems: "center", justifyContent: "space-between", padding: "6px 12px", borderBottom: "1px solid #3A3A3A" }}>
        <span style={{ fontSize: 12, color: "rgba(255,255,255,0.4)" }}>
          {filePath || "未选择文件"}
        </span>
        <button
          disabled={!dirty}
          onClick={handleSave}
          style={{
            padding: "3px 10px",
            background: dirty ? "rgba(110,231,183,0.12)" : "transparent",
            border: `1px solid ${dirty ? "rgba(110,231,183,0.3)" : "rgba(255,255,255,0.06)"}`,
            borderRadius: 4,
            color: dirty ? "#6ee7b7" : "rgba(255,255,255,0.3)",
            fontSize: 11,
            cursor: dirty ? "pointer" : "default",
          }}
        >
          保存
        </button>
      </div>
      <div style={{ flex: 1, overflow: "auto", padding: 8 }}>
        <MilkdownProvider>
          <EditorInner content={content} onChange={handleChange} />
        </MilkdownProvider>
      </div>
    </div>
  );
}
```

- [ ] **Step 10: Write ScheduledTasksSection.tsx**

```tsx
import { useState } from "react";
import { useSettingsStore, type ScheduledTask } from "../../stores/settingsStore";
import styles from "./SettingsPage.module.css";

function genId() {
  return Date.now().toString(36) + Math.random().toString(36).slice(2);
}

export function ScheduledTasksSection() {
  const tasks = useSettingsStore((s) => s.scheduledTasks);
  const updateScheduledTask = useSettingsStore((s) => s.updateScheduledTask);
  const setScheduledTasks = useSettingsStore((s) => s.setScheduledTasks);
  const [showAdd, setShowAdd] = useState(false);
  const [newTask, setNewTask] = useState({ name: "", cronExpression: "", action: "reindex" });

  const handleAdd = () => {
    if (!newTask.name.trim()) return;
    const task: ScheduledTask = {
      id: genId(),
      name: newTask.name,
      cronExpression: newTask.cronExpression || "0 3 * * *",
      action: newTask.action,
      enabled: true,
    };
    setScheduledTasks([...tasks, task]);
    setNewTask({ name: "", cronExpression: "", action: "reindex" });
    setShowAdd(false);
  };

  const handleDelete = (id: string) => {
    setScheduledTasks(tasks.filter((t) => t.id !== id));
  };

  return (
    <div>
      <h2 className={styles.sectionTitle}>定时任务</h2>
      <div className={styles.desc} style={{ marginBottom: 16 }}>
        配置自动执行的后台任务，如知识库重建索引、数据备份、同步等。
      </div>

      {tasks.length === 0 && !showAdd && (
        <div style={{ color: "rgba(255,255,255,0.25)", fontSize: 12, marginBottom: 16 }}>
          暂无定时任务
        </div>
      )}

      <div style={{ display: "flex", flexDirection: "column", gap: 8, marginBottom: 16 }}>
        {tasks.map((task) => (
          <div
            key={task.id}
            style={{
              display: "flex",
              alignItems: "center",
              justifyContent: "space-between",
              padding: "10px 12px",
              background: "rgba(255,255,255,0.03)",
              border: "1px solid rgba(255,255,255,0.06)",
              borderRadius: 6,
            }}
          >
            <div>
              <div style={{ color: "#e0e0e0", fontSize: 13 }}>{task.name}</div>
              <div style={{ color: "rgba(255,255,255,0.3)", fontSize: 11, marginTop: 2 }}>
                {task.cronExpression} | 操作: {task.action} |
                状态: {task.enabled ? "启用" : "禁用"}
                {task.lastRun && ` | 上次: ${task.lastRun}`}
              </div>
            </div>
            <div style={{ display: "flex", gap: 6, alignItems: "center" }}>
              <button
                className={`${styles.toggleSwitch} ${task.enabled ? styles.toggleOn : styles.toggleOff}`}
                style={{ width: 36, height: 20, borderRadius: 10 }}
                onClick={() => updateScheduledTask(task.id, { enabled: !task.enabled })}
              >
                <div
                  className={styles.toggleKnob}
                  style={{
                    width: 16, height: 16,
                    transform: task.enabled ? "translateX(16px)" : "translateX(2px)",
                  }}
                />
              </button>
              <button
                style={{
                  padding: "3px 8px",
                  background: "transparent",
                  border: "1px solid rgba(255,100,100,0.25)",
                  borderRadius: 4,
                  color: "#ff6464",
                  fontSize: 10,
                  cursor: "pointer",
                }}
                onClick={() => handleDelete(task.id)}
              >
                删除
              </button>
            </div>
          </div>
        ))}
      </div>

      {showAdd && (
        <div
          style={{
            padding: 12,
            background: "rgba(255,255,255,0.02)",
            border: "1px solid rgba(110,231,183,0.15)",
            borderRadius: 6,
            marginBottom: 12,
          }}
        >
          <div className={styles.fieldGroup}>
            <label className={styles.label}>任务名称</label>
            <input
              className={styles.textInput}
              value={newTask.name}
              onChange={(e) => setNewTask({ ...newTask, name: e.target.value })}
              placeholder="例如：每日知识库重建索引"
            />
          </div>
          <div className={styles.fieldGroup}>
            <label className={styles.label}>Cron 表达式</label>
            <input
              className={styles.textInput}
              value={newTask.cronExpression}
              onChange={(e) => setNewTask({ ...newTask, cronExpression: e.target.value })}
              placeholder="0 3 * * *（每天凌晨3点）"
              style={{ maxWidth: 300 }}
            />
          </div>
          <div className={styles.fieldGroup}>
            <label className={styles.label}>操作类型</label>
            <select
              className={styles.select}
              value={newTask.action}
              onChange={(e) => setNewTask({ ...newTask, action: e.target.value })}
            >
              <option value="reindex">重建知识索引</option>
              <option value="backup">数据备份</option>
              <option value="sync">同步知识库</option>
              <option value="cleanup">清理旧数据</option>
            </select>
          </div>
          <div style={{ display: "flex", gap: 8 }}>
            <button className={styles.btnPrimary} onClick={handleAdd}>添加</button>
            <button
              style={{
                padding: "7px 16px",
                background: "transparent",
                border: "1px solid rgba(255,255,255,0.1)",
                borderRadius: 5,
                color: "rgba(255,255,255,0.4)",
                fontSize: 12,
                cursor: "pointer",
              }}
              onClick={() => setShowAdd(false)}
            >
              取消
            </button>
          </div>
        </div>
      )}

      {!showAdd && (
        <button className={styles.btnPrimary} onClick={() => setShowAdd(true)}>
          + 添加任务
        </button>
      )}
    </div>
  );
}
```
