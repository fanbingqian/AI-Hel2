# 快速开始
AI-Hel2 是一个桌面端 AI 助手，核心交互界面为 3D 知识图谱球体 + 语音对话。
核心依赖 @Hermes Agent 进行对话推理和知识管理。
通过 @知识图谱 实现语义搜索与可视化。
支持 @Tauri 2.0 桌面框架打包，跨平台运行。
## 安装步骤
1. 下载对应平台的安装包
2. 解压并运行 AI-Hel2.exe
3. 首次启动会自动初始化知识库
## 语音唤醒
说 "Hi Hel" 唤醒语音助手，开始对话。
